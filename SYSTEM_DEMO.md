# TOPO系统 - 系统演示指南

## 演示概述

本演示将展示TOPO系统的核心功能，包括项目管理、Bug跟踪、用户管理和统计分析。演示时长约30分钟。

## 演示准备

### 系统环境
- 确保后端服务运行在 http://localhost:5000
- 确保前端服务运行在 http://localhost:3001
- 准备测试数据（项目、Bug、用户）

### 演示账户
- 管理员：admin / admin123
- 项目经理：project_manager / project123
- 开发人员：developer1 / dev123
- 测试人员：tester1 / test123

## 演示流程

### 第一部分：系统介绍（5分钟）

#### 1.1 登录系统
- 打开浏览器访问 http://localhost:3001
- 使用管理员账户登录
- 展示系统界面布局

#### 1.2 仪表板概览
- 展示项目统计信息
- 展示Bug状态分布
- 展示系统健康状态

### 第二部分：项目管理演示（10分钟）

#### 2.1 创建新项目
- 点击"项目管理" → "新建项目"
- 填写项目信息：
  - 项目名称："演示项目"
  - 描述："系统功能演示和培训用项目"
  - 状态：active
  - 成员：选择项目经理、开发人员、测试人员
- 演示项目创建成功

#### 2.2 项目编辑和成员管理
- 编辑项目信息
- 添加/移除项目成员
- 演示权限控制

#### 2.3 项目筛选和搜索
- 按状态筛选项目
- 按成员筛选项目
- 搜索功能演示

### 第三部分：Bug管理演示（10分钟）

#### 3.1 创建Bug报告
- 切换到测试人员账户
- 创建新的Bug报告：
  - 标题："购物车页面加载缓慢"
  - 描述：详细描述问题现象和重现步骤
  - 严重程度：高
  - 优先级：高
  - 分配人员：developer1
- 演示附件上传功能

#### 3.2 Bug状态流转
- 切换到开发人员账户
- 查看分配的Bug
- 将Bug状态改为"进行中"
- 完成修复后改为"已解决"

#### 3.3 Bug验证和关闭
- 切换回测试人员账户
- 验证Bug修复情况
- 关闭Bug或重新打开

#### 3.4 Bug筛选和搜索
- 按状态、严重程度、优先级筛选
- 高级搜索功能
- 批量操作演示

### 第四部分：高级功能演示（5分钟）

#### 4.1 数据导出
- 导出项目列表（CSV/Excel）
- 导出Bug数据
- 演示导出文件内容

#### 4.2 统计分析
- 展示Bug趋势图表
- 项目进度分析
- 团队工作效率统计
