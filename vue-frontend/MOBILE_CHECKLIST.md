## 🔧 CSS工具类使用指南

### 基础工具类
```vue
<div class="hidden-mobile">桌面端显示</div>
<div class="hidden-desktop">移动端显示</div>
<div class="flex-mobile-column">移动端垂直布局</div>
<div class="w-full">100%宽度</div>
```

### 间距工具类
```vue
<div class="m-4">移动端边距16px</div>
<div class="p-3">移动端内边距12px</div>
<div class="mx-auto">水平居中</div>
```

### 文本工具类
```vue
<span class="text-xs">12px文字</span>
<span class="text-center">居中文字</span>
<span class="text-truncate">超长省略</span>
```

### 卡片工具类
```vue
<div class="mobile-card">
  <div class="mobile-card-header">头部</div>
  <div class="mobile-card-body">内容</div>
  <div class="mobile-card-footer">底部</div>
</div>
```

## 🎨 设计原则

1. **渐进增强** - 先保证基本功能，再提升体验
2. **移动优先** - 优先考虑移动端体验
3. **性能优先** - 减少不必要动画和资源
4. **触摸友好** - 足够大的点击区域
5. **内容为王** - 确保内容优先展示
