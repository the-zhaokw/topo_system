# TOPO系统微信小程序 - 使用指南

## 📋 前置准备

### 1. 安装 Node.js
- 下载地址：https://nodejs.org/
- 推荐版本：v18.x 或更高版本
- 验证安装：打开终端，输入 `node -v`

### 2. 安装微信开发者工具
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 推荐版本：稳定版最新版
- 安装后登录微信开发者账号

---

## 🚀 项目配置步骤

### 第一步：安装依赖

打开终端，进入项目目录：

```bash
cd d:\topo_system\mini-app
```

安装项目依赖：

```bash
npm install
```

等待安装完成，这可能需要几分钟时间。

### 第二步：编译项目

编译微信小程序版本：

```bash
npm run build:weapp
```

或者使用开发模式（文件修改自动重新编译）：

```bash
npm run dev:weapp
```

### 第三步：打开微信开发者工具

1. 打开微信开发者工具
2. 点击左上角的"导入项目"
3. 项目目录选择：`d:\topo_system\mini-app\dist`
4. AppID 输入：
   - 如果有正式小程序账号，输入 AppID
   - 如果没有，可以输入测试号：`wx0000000000000000`（仅供测试）
5. 项目名称：`TOPO系统小程序`
6. 点击"确认"

### 第四步：配置编译设置

在微信开发者工具中：

1. 点击右上角"详情"
2. 选择"本地设置"
3. 勾选：
   - ✅ 使用 npm 模块
   - ✅ 不校验合法域名（开发环境）
4. 编译模式选择："多页模式"

### 第五步：安装npm依赖（微信开发者工具内）

1. 在微信开发者工具中，点击顶部菜单"工具"
2. 选择"构建 npm"
3. 等待构建完成

### 第六步：预览项目

1. 点击微信开发者工具顶部的"编译"按钮
2. 或者按快捷键 `Ctrl + B`
3. 等待编译完成，即可在模拟器中看到小程序

## ⚠️ 常见问题

### 1. npm install 失败

**问题**：安装依赖时出错

**解决方案**：
```bash
# 清除缓存
npm cache clean --force

# 删除 node_modules
rm -rf node_modules

# 重新安装
npm install
```

### 2. 编译报错

**问题**：编译时报错

**解决方案**：
```bash
# 清理构建缓存
rm -rf dist
rm -rf node_modules/.cache

# 重新编译
npm run build:weapp
```

### 3. 微信开发者工具无法导入

**问题**：导入项目时提示"无效的目录"

**解决方案**：
1. 确保已经运行 `npm run build:weapp`
2. 检查 `dist` 目录是否存在
3. 确保 `app.json` 文件存在

### 4. npm 模块找不到

**问题**：运行时报错找不到模块

**解决方案**：
1. 在微信开发者工具中执行"构建 npm"
2. 确保 package.json 中包含该模块

### 5. 样式不生效

**问题**：修改样式后没有变化

**解决方案**：
1. 点击微信开发者工具的"编译"按钮
2. 或者按 `Ctrl + B` 重新编译
3. 清除缓存：`rm -rf dist && npm run build:weapp`

## 🔧 高级配置

### 修改编译输出目录

如果需要修改编译输出目录，编辑 `config/index.js`：

```javascript
weapp: {
  outputRoot: 'dist-weapp',  // 修改这里
  // ...
}
```

### 配置 ESLint

项目已配置 ESLint，修复错误：

```bash
npm run lint
```

### TypeScript 类型检查

```bash
npx tsc --noEmit
```
