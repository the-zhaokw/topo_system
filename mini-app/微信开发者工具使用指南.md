## 🔧 方法二：使用微信开发者工具

### 第一步：安装微信开发者工具

1. 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 安装并登录您的微信账号

### 第二步：清理并重新生成项目

由于之前的操作导致SCSS文件损坏，让我们重新初始化：

```bash
# 进入项目目录
cd d:\topo_system\mini-app

# 删除损坏的node_modules（如果需要）
rm -rf node_modules

# 重新安装依赖
npm install

# 重新初始化模板
node "c:\Users\DELL\.trae-cn\builtin_skills\TRAE-generate-mini-app\scripts\init-template.js" "d:\topo_system\mini-app-temp"
```

### 第三步：复制核心文件

从您之前创建的项目中复制必要的文件：
- `/src/pages/home/` - 首页
- `/src/pages/bug-list/` - Bug列表
- `/src/pages/bug-detail/` - Bug详情
- `/src/pages/project-list/` - 项目列表
- `/src/pages/project-detail/` - 项目详情
- `/src/pages/my-todos/` - 我的待办
- `/src/pages/attendance/` - 考勤打卡
- `/src/pages/contracts/` - 合同列表
- `/src/pages/materials/` - 物料列表
- `/src/pages/knowledge/` - 知识库
- `/src/pages/notifications/` - 通知列表
- `/src/pages/personal-plan/` - 个人计划
- `/src/pages/mine/` - 我的

### 第四步：编译并打开

```bash
# 编译微信小程序版本
npm run build:weapp
```

### 第五步：导入微信开发者工具

1. 打开微信开发者工具
2. 点击"导入项目"
3. 项目目录选择：`d:\topo_system\mini-app\dist`
4. AppID：可以输入测试号 `wx0000000000000000`
5. 项目名称：`TOPO系统小程序`
6. 点击"确认"

### 第六步：配置并运行

1. 在详情 → 本地设置中：
   - ✅ 勾选"使用 npm 模块"
   - ✅ 勾选"不校验合法域名"
2. 点击顶部菜单"工具" → "构建 npm"
3. 点击"编译"按钮
4. 在模拟器中查看效果

## ❓ 常见问题

### Q: 微信开发者工具导入失败？
**A**: 确保已经运行 `npm run build:weapp`，并且 `dist` 目录存在。

### Q: 编译报错？
**A**: 尝试清理并重新安装：
```bash
rm -rf node_modules dist
npm install
npm run build:weapp
```

### Q: 如何更新代码？
**A**: 
1. 修改源代码
2. 运行 `npm run build:weapp`
3. 在微信开发者工具中点击"编译"
